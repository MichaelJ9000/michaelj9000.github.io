C++学习笔记

# 基本内置类型

# 算术类型

  1. 当明确知道数值不可能为负时，使用无符号类型
  2. 使用int进行整数运算，short太小，long的尺寸和int一样，如果超出了int的表示范围，使用long long
  3. 在算术运算中不使用char和bool
  4. 执行浮点数的时候用double

# 类型int、long、long long和short的区别是什么？

C++规定int和short最小是16位，long最小是32位。long long最小是64位

# 无符号类型和带符号类型的区别是什么?float和double的区别是什么?

无符号类型能表示0和正数，带符号类型还能表示负数。float和double分别能表示7和16位有效数字，用32位或者64位表示

# 类型转换

不要混用有符号数和无符号数

# 字面值常量

一个形如42的值被称做字面值常量。每一个字面值常来那个都对应一种数据类型，字面值常量的值和形式决定了它的数据类型。

# 字符和字符串字面值

'a'  //字符字面值
"hello world"  //字符串字面值
1
2

字符串字面值的类型实际上是常量字符构成的数组(array)

编译器会在每个字符串结尾加一个空字符('\0')

因此字符串字面量的实际长度要比它的内容多1。

# 变量

变量提供一个具名的、可操作的存储空间。C++的每个变量都有其数据类型，数据类型决定着变量所占的内存空间大小和分布方式、该空间能存储值的范围以及变量能参与的运算。对C++程序员来说变量(variable)和对象(object)一般能够交互使用。
